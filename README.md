# OOP_CPP_projects

This repository contains a series of practical exercises and laboratory works focused on mastering Object-Oriented Programming in C++. It serves as a comprehensive practice ground for implementing core programming patterns and data structures.

## 📁 Practice Modules

The repository covers the following C++ implementations:

* **Laboratory Works** – Structured tasks covering fundamental to advanced C++ concepts.
* **Data Structures** – Custom implementations of stacks, queues, and linked lists.
* **Class Hierarchies** – Examples of inheritance and polymorphism in real-world scenarios.
* **Memory Management** – Practical use of pointers, dynamic allocation, and destructors.
* **Operator Overloading** – Enhancing custom classes with intuitive mathematical and I/O operations.

## 🛠 Tech Stack
* **Language:** C++
* **Environment:** GCC and G++
* **Focus:** Logic, Efficiency, and OOP Principles
